# Problem Set 2: Bank Term Deposit Prediction using Logistic Regression

## Problem Statement

A banking institution wants to predict whether a customer will subscribe to a term deposit (`yes` or `no`) based on their demographic information and banking behavior using **Logistic Regression**.

## Dataset Information

* **Dataset**: Bank Marketing Dataset (`bank-full.csv`)
* **Number of rows**: 45,211
* **Number of columns**: 17
* **Target variable**: `y` (yes = subscribed to term deposit, no = not subscribed)
* **Type**: Binary Classification

**Key Features**:

* age, job, marital, education, default, balance, housing, loan, contact, day, month, duration, campaign, pdays, previous, poutcome
